                          Der Curses-Minesweeper

Das Minesweeper-Spielbrett enthält eine Anzahl versteckter Minen.  Bei jedem
Zug decken Sie eines oder mehrere Felder auf, um ihren Inhalt zu erfahren.  Um
zu gewinnen, müssen Sie die Position jeder Mine ableiten, ohne eine aufzu-
decken.  Sie können "Fahnen" auf dem Brett platzieren, um nicht versehentlich
auf eine bekannte Mine zu treten;  Felder können nicht aufgedeckt werden,
solange eine Fahne darauf steht.  Sie gewinnen, sobald alle nicht verminten
Felder aufgedeckt sind.

Wenn Sie ein vermintes Feld aufdecken, verlieren Sie sofort.  Decken Sie ein
freies Feld auf, so erscheint sein Inhalt:  Eine Zahl, die angibt, wieviele
Minen auf den angrenzenden acht Feldern liegen.  Sind keine Minen im Umkreis
des Feldes, werden auch die umgebenden Felder automatisch aufgedeckt.

Um den Cursor auf dem Brett zu bewegen, verwenden Sie die Pfeiltasten.  "f"
setzt oder entfernt eine Fahne am Cursor.  Die Eingabetaste bzw. "Enter" deckt
ein Feld auf.  Drücken Sie auf einem bereits aufgedeckten Feld nochmals
"Enter", so werden alle umgebenden Felder aufgedeckt, wenn um den Cursor genau
so viele Fahnen gesetzt wie Minen versteckt sind.  (Wenn Sie einen Fehler beim
Setzen der Fahnen gemacht haben, werden Sie dadurch verlieren.)

"n" beginnt ein neues Spiel.  Spiele können mit "s" und "l" gespeichert bzw.
geladen werden.  Dazu müssen Sie jeweils einen Dateinamen zum Speichern bzw.
Laden angeben.

Zum Beenden von Minesweeper drücken Sie bitte "q".
